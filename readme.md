# Changes

**Changes** is a parkour-puzzle game built in GoDot 4.6. Explore different worlds with a sarcastic narrator, solve levels testing your patience and skills and fight "some random aliens".

## Structure
- `project/scenes/`: The levels that are going to be played.
- `project/scripts/`: Main logic for the game that includes player control, level stuff and more.
- `project/assets/`: Assets for stuff like sound/music, tilesets and more.
